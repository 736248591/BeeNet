# user 用户模块

## get_login_token 获取登录加密令牌

**描述：** 获取用于加密的令牌。

**参数：** （无）

**响应数据：**
```
{
   "status": 操作状态 Boolean,
   "mgs": 错误原因 (当状态为false时，拥有这个字段）string,
   "token": 盐
 }
```

## login 登录

**描述**：输入用户密码，时间（毫秒）/1000插入密码的左右，做一次MD5。在结果两段加上之前获取的盐，做一次SHA3。

**参数：**

| **字段名** | **数据类型** | **默认值** | **描 述** |
| :----------: | :------------: | :----------: | :---------: |
| pwd     | string |            | 密码   |

**响应数据：**

```
{
   "status": 操作状态 Boolean,
   "mgs": 错误原因 (当状态为false时，拥有这个字段）string
 }
```
